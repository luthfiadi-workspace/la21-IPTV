## 🏆 World Cup 2026

`world-cup.m3u` hanya memuat broadcaster resmi free-to-air yang benar-benar bisa diputar tanpa batasan geo (BBC One/Two, ČT Sport, M6, Rai Sport, La 1, RTP 1, TVP Sport, TSN, TV Azteca, Rede Globo/CazeTV, TVRI, KBS World, NHK World-Japan, Al Jazeera English). Broadcaster berbayar (beIN Sports, Zee Sports, SuperSport) dan broadcaster yang tidak punya stream gratis publik (BBC iPlayer asli, ITV, ARD, ZDF, RT, CCTV, MBC/SBS Korea, VRT) tidak disertakan. Untuk siaran olahraga umum non-Piala Dunia, lihat `sports.m3u`.

## 🇮🇩 Ranking Channel Indonesia

`indonesia.m3u` diurutkan manual berdasarkan popularitas free-to-air: RCTI, SCTV, Indosiar, Trans TV, Trans7, ANTV, MNCTV, GTV, tvOne, Metro TV, Kompas TV, NET, iNews, TVRI, lalu sisanya. Catatan: SCTV, Indosiar, GTV, Kompas TV, dan NET saat ini **tidak tersedia** sebagai stream gratis publik di sumber data (iptv-org) — kemungkinan dibatasi hak siar.

## 📱 Cara Pakai di TiviMate / Player IPTV lain

1. Copy URL playlist dari tabel di atas
2. Buka TiviMate → Settings → Playlists → Add Playlist → M3U URL → Paste URL
3. Channel otomatis terkelompok per kategori, lengkap dengan logo dan EPG (jika tersedia)

---

## 🔄 Auto Update

Playlist diperbarui otomatis setiap hari **pukul 08.00 WIB** via GitHub Actions dari sumber [iptv-org](https://github.com/iptv-org/iptv).
