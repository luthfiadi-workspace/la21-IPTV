📱 Cara Pakai di BrowseHere (Google TV)
1
Copy URL playlist
Klik tombol "Copy URL" di salah satu playlist di atas. Untuk semua channel, gunakan Full Playlist.
https://raw.githubusercontent.com/luthfiadi-workspace/la21-IPTV/main/playlists/full.m3u
2
Buka BrowseHere di Google TV
Buka aplikasi BrowseHere → pilih menu IPTV atau M3U Player → pilih "Add Playlist".
3
Paste URL & Load
Paste URL yang sudah dicopy → klik Load/OK. Channel akan muncul terkelompok per kategori otomatis.
4
Update Otomatis
Playlist diperbarui otomatis setiap hari pukul 08.00 WIB via GitHub Actions. Tidak perlu update manual.
